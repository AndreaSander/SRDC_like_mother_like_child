This is the code for the manuscript Like Mother Like Child: Differential Impact of Mothers’ and Fathers’ Individual Language Use on Bilingual Language Exposure.

The first Rmd file (load merge) cleans the anonymized archival data. The second Rmd file wrangles the data to generate all the results reported in the manuscript and supplementary materials. All the manuscript figures and tables have been generated and saved into this project's folder.
